3.1.2 Ussaṅkitasutta
Tatiyapaṇṇāsaka
Phāsuvihāravagga
Ussaṅkitasutta
102.
615“Pañcahi, bhikkhave, dhammehi samannāgato bhikkhu ussaṅkitaparisaṅkito hoti pāpabhikkhūti api akuppadhammopi.
616Katamehi pañcahi? Idha, bhikkhave, bhikkhu vesiyāgocaro vā hoti, vidhavāgocaro vā hoti, thullakumārikāgocaro vā hoti, paṇḍakagocaro vā hoti, bhikkhunīgocaro vā hoti.
617Imehi kho, bhikkhave, pañcahi dhammehi samannāgato bhikkhu ussaṅkitaparisaṅkito hoti pāpabhikkhūti api akuppadhammopī”ti.
618Dutiyaṃ.
