3.4.6 Abhinandasutta
Dhātusaṃyutta
Catutthavagga
Abhinandasutta
119.
809Sāvatthiyaṃ viharati. “Yo, bhikkhave, pathavīdhātuṃ abhinandati, dukkhaṃ so abhinandati. Yo dukkhaṃ abhinandati, aparimutto so dukkhasmāti vadāmi. Yo āpodhātuṃ abhinandati…pe…  yo tejodhātuṃ…  yo vāyodhātuṃ abhinandati, dukkhaṃ so abhinandati. Yo dukkhaṃ abhinandati, aparimutto so dukkhasmāti vadāmi.
810Yo ca kho, bhikkhave, pathavīdhātuṃ nābhinandati, dukkhaṃ so nābhinandati. Yo dukkhaṃ nābhinandati, parimutto so dukkhasmāti vadāmi. Yo āpodhātuṃ…pe…  yo tejodhātuṃ…  yo vāyodhātuṃ nābhinandati, dukkhaṃ so nābhinandati. Yo dukkhaṃ nābhinandati, parimutto so dukkhasmāti vadāmī”ti.
811Chaṭṭhaṃ.
