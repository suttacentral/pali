4.1.10 Seyyajātaka
Catukkanipāta
Kāliṅgavagga
Seyyajātaka
37.
928
“Sasamuddapariyāyaṃ,
mahiṃ sāgarakuṇḍalaṃ;
Na icche saha nindāya,
evaṃ seyya vijānahi.
38.
929
Dhiratthu taṃ yasalābhaṃ,
dhanalābhañca brāhmaṇa;
Yā vutti vinipātena,
adhammacaraṇena vā.
39.
930
Api ce pattamādāya,
anagāro paribbaje;
Sāyeva jīvikā seyyo,
yā cādhammena esanā.
40.
931
Api ce pattamādāya,
anagāro paribbaje;
Aññaṃ ahiṃsayaṃ loke,
api rajjena taṃ varan”ti.
932Seyyajātakaṃ dasamaṃ.
Kāliṅgavaggo paṭhamo.
933Tassuddānaṃ
934
Vivarañca adeyya samiddhavaraṃ,
Atha daddara pāpamahātiraho;
Atha koli palāsavarañca kara,
Carimaṃ sasamuddavarena dasāti.
